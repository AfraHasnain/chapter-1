HTML

Hypertext Markup Language

Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphics and hyperlink effects on world wide web pages.

CSS

Cascading Style Sheet

CSS describes how HTML element are to displayed on screen, paper or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once. External Stylesheets are stored in CSS files.

JS
Java Script

JavaScript is a scripting language used to create and control dynamic website content, i.e. anything that moves, refreshes, or otherwise changes on your screen without requiring you to manually reload a web page. Features like: animated graphics. photo slideshows.
